AI Sort Guide

What is AI Sort?

AI Sort uses an AI chatbot (like Claude or ChatGPT) to automatically sort your mods into the correct Kenshi load order. It generates a text file with all your mod descriptions, which you paste into an AI chat. The AI returns a sorted list you can import back.

How it works

  1. The app calls Steam's public API to fetch the full Workshop description for each mod that needs sorting.
  2. It builds a text file containing each mod's name, filename, and description, along with your categories and a link to the Kenshi load order guide.
  3. You paste this into an AI chat. The AI reads every mod description, references the load order guide, and figures out which category each mod belongs in.
  4. The AI responds with a sorted list. You paste that back into the app and it applies the new order.

The app never contacts any AI service directly — you are the middleman. Nothing leaves your machine until you manually paste it into a chat.

How to use it

  1. Click "AI Sort Export" — this fetches full Steam descriptions for your uncategorized mods only and includes your current sorted categories as context. Already-sorted mods aren't re-described, saving tokens.
  2. Open an AI chatbot — Claude (claude.ai) is recommended for large mod lists (supports ~200k tokens). ChatGPT works for smaller lists.
  3. Paste the entire contents of the exported .txt file into the AI chat and send it.
  4. Copy the AI's response — it should be a clean list of [Category] headers and mod filenames.
  5. Click "AI Sort Import" — paste the AI's response and click "Apply Order".

Important notes

  • Save a mod pack first! Use the Packs tab to save your current setup before importing an AI sort. This way you can always revert.
  • The AI's sorting is a suggestion, not gospel. Always review the result and adjust manually if needed.
  • The AI may occasionally put a mod in the wrong category — it's working from descriptions alone and doesn't understand every mod's internals.
  • For very large mod lists (500+), the AI may truncate its response. If mods are missing after import, they'll stay in their current category.

What the AI response should look like

The AI should respond with ONLY this format — no intro text, no notes:

[UI, Graphics, Performance]
Dark UI.mod
Nice Map [Zones + Zone names + Roads].mod
Water Optimizer.mod

[Animations]
AnimationOverhaul.mod
More Combat Animation.mod

[New Races & Race Edits]
Race Blood.mod

[Animals]
TameBeasties.mod

[Uncategorized]
Test.mod

If the AI adds extra commentary, that's OK — the parser strips everything that isn't a [Category] header or a .mod filename.

Is this safe?

  • No mod files are uploaded. Only mod names and their public Steam Workshop descriptions are included in the export file.
  • One API call to Steam. The app makes a single bulk request to Steam's public API to fetch descriptions. This is the same API used by every mod manager and is well within rate limits.
  • Nothing is sent automatically. You manually paste the file into an AI chat of your choice. The app never contacts any AI service directly.
  • Your load order is only changed when you click "Apply Order" in the import dialog. Nothing is modified until you confirm.
AI Sort Export
Uncategorized Only (Recommended)

Only fetches descriptions for mods in Uncategorized. Already-sorted mods are listed by filename only. Much smaller file, faster AI processing, lower cost.

Uncategorized Compact (Recommended for 50+ unsorted)

Only unsorted mods, using tags + truncated descriptions. Already-sorted mods listed by filename. Best balance of accuracy and size.

All Mods — Compact

All mods using Steam tags + first 200 characters. Fits 500+ mods in a single prompt. Use for a full re-sort without exceeding token limits.

All Mods — Full Descriptions

Fetches full descriptions for every active mod. Most accurate but produces a very large file. May exceed AI token limits for 100+ mods.

Split into Batches

Exports multiple files with ~50 mods each (full descriptions). Run each through the AI separately. Most accurate but most tedious.

Note: You have a large mod list. "Compact" mode is recommended — it fits everything in one prompt. "Full Descriptions" may exceed AI token limits.
